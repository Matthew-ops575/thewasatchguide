## Competitor Landscape

### National Aggregators (Positions 1-3 for most queries)

| Site | Strengths | Vulnerability |
|---|---|---|
| Niche.com | Massive DA, ranks for every "living in" query | Generic data, no local voice, no photography |
| BestPlaces.net | Owns comparison queries | Pure data tables, no narrative |
| Livability.com | Top-5 for "moving to" queries | Template-driven, thin profiles |
| AreaVibes.com | Cost of living scores | Data-only |
| Homes.com | City guide pages | Real estate portal, not editorial |

| trupprealestategroup.com | Cottonwood Heights | Single post, no depth |

### Service Directories (Competitors for vertical keywords)

| Site | Position |
|---|---|
| Yelp | #1-3 for virtually every service query |
| Angi | Top-3 for home services |
| HomeAdvisor | Strong for plumbing, electrical, roofing |
| Healthgrades | Dominates medical specialist queries |

### The Wasatch Guide's Competitive Advantage

No independent editorial publication covers the Wasatch Front neighborhood-by-neighborhood. Every competitor is either a national aggregator (generic data), a real estate agent blog (commercial intent), or a directory (listings, not editorial). The Wasatch Guide occupies the gap: independent local editorial with depth, local photography, and genuine voice.

## Content Format Notes

**Pillar guides** (2,000–2,500 words): Target "living in [city] utah" + sub-keywords as H2/H3 sections. Each guide should cover: overview/vibe, schools, real estate, commute, neighborhoods, lifestyle/outdoors, market snapshot.

**Comparison posts** (1,000–1,500 words): Target "[city] vs [city] utah". Side-by-side with honest trade-offs. Link back to both pillar guides.

**Service roundups** (1,000–1,500 words): Target "best [service] in [city] utah". Curated editorial format — not a directory listing. Name 5-7 providers with honest assessments. This is the content type that converts to consulting clients.

**Seasonal posts** (600–1,200 words): Target trending/seasonal queries. Refresh annually with updated data. Link to relevant pillar guides and verticals.
